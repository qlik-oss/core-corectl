## corectl variable rm

Remove one or many variables in the current app

### Synopsis

Remove one or many variables in the current app

```
corectl variable rm <variable-name>... [flags]
```

### Examples

```
corectl variable rm NAME-1
```

### Options

```

      --no-save   Do not save the app
```
